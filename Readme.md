# Unforgivable.DB2.Editor

Лёгкий C#‑редактор для DB2 файлов World of Warcraft, заточенный под TrinityCore и кастомный моддинг клиента. [web:216][web:219]

## Возможности

- Открытие и просмотр DB2 таблиц в табличном виде. [web:216]
- Редактирование ячеек с сохранением обратно в DB2. [web:216]
- Поиск и фильтрация строк (удобно для DisplayID, SpellID, RaceID и т.п.).
- Импорт/экспорт строк в CSV для массового редактирования.
- Поддержка нескольких версий клиентских DB2 (зависит от настроек проекта).

## Установка

1. Скачайте последний релиз из раздела Releases этого репозитория. [web:214]
2. Распакуйте архив в любую папку.
3. Запустите `Unforgivable.DB2.Editor.exe`.

## Сборка из исходников

Требуется:

- Visual Studio 2022 или новее.
- .NET SDK подходящей версии (см. `Unforgivable.DB2.Editor.csproj`). [web:216]

Шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/<ваш-ник>/Unforgivable.DB2.Editor.git
   ```
2. Откройте решение в Visual Studio.
3. Соберите конфигурацию `Release`.
4. Готовые бинарники появятся в `bin/Release/netX.X/`.

## Использование

- Файл → Open DB2… — выберите нужный DB2 из клиентской папки.
- Измените нужные значения в таблице.
- Нажмите Save, чтобы перезаписать DB2.
- Для бэкапа можно использовать Save As и сохранить копию отдельно.

Пример:  
Меняем DisplayID кастомной расы — находим строку по ID, правим DisplayID, сохраняем DB2 и перезапускаем клиент.
